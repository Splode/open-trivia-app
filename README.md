## About
A trivia web app built with Vue.js and powered by local files.

State is managed by [Vuex](https://github.com/vuejs/vuex). Database requests are handled by [vue-resource](https://github.com/pagekit/vue-resource).

## Use

```npm install```
```npm run build```
